# 东方山寨（未完成）
这是一个毫无创意的打飞机游戏引擎。

特点：
* **渣优化**：计算量是同类游戏的4倍，需要性能较高的中央处理器和显卡才能流畅运行。
* **难以阅读**：全世界约五分之四的人连代码里的文字都看不懂。
